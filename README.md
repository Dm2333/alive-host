AliveHost is a simple bash script that takes a list of hosts (Ex: from Sublist3r) and checks to see if they're up / alive!

## Installion / Usage
* You need Linux / Unix
* You need to install nmap 
*  `chmod +x alive.sh`
* Usage: ./alive.sh HostList.txt
